# git challenge #11 に参加した
#### 2019-05-18

株式会社ミクシィが主催する git challenge の第11回目に参加してきました。  
  
  
<Microlink url={"https://mixi-git-challenge.github.io/publications/events/11.html"} style={microLinkStyle}/>
  

git challenge というイベントの存在自体は少し前から耳にしていて、前回は都合がつかず応募できなかったのですが、
今回応募に通り無事参加することができました。

---
  
git challenge では2人1組でペアになって git に関する問題をひたすら解きます。  
最初の方は落ち着いて問題文の指示に従えば解けるような問題ですが、後半に進むにつれて、あれこれ捻った問題が出されます。  
私は最初の方の問題は割合すんなり正解できましたが、後半のある1問でかなり詰まってしまいました（しかし後で解説を聞くと、あるコマンドを使えば簡単に処理が済むことが分かりました）。  
私のペアの相方はとても優秀な方で、私より沢山の問題を素早く解いてくれました。  
結果としては、得点数では1位（同率で1位が3チームいました）になりましたが、1位の得点になるまでの時間で他のチームに及ばず、優勝できませんでした。  
  
git challenge の問題で出てくるシチュエーションは、実際の開発で起こってほしくないような恐ろしいものばかりです。
適切な git の運用と、いざという時にきちんとリポジトリを復旧できるような対応力を普段から心がけたいものです。
  
---
  
その後は、おいしいご飯を食べたりお酒を飲んだり、他の学生やエンジニアの方と雑談をしたりしていました。ミクシィさん楽しいイベントありがとうございました。
