# ブロック一括解除 for Twitter Lite
#### 2020-01-29

「Twitter ブロック 一括解除」でググると、上位にこの記事が出てきます。

<Microlink url={"https://www.ncaq.net/2017/08/21/"} style={microLinkStyle}/>

ここに書かれてある通り、デベロッパーツールでブロック解除ボタンをクリックするスクリプトを書くのが早いです。
ただ、記載されているコードが今の Twitter（Twitter Lite と呼ばれる PWA）では使えませんでした。  
  
そこで、2020年1月現在使えるコードを代わりにシェアします。

```js
document.querySelectorAll("div[aria-label=\"ブロック中\"]").forEach(b => b.click());
```

これもサイトの構造が変化したら使えなくなるかもしれません。もし使えなくて困っている方がいましたら、
[Issue](https://github.com/7ma7X/hellorusk.net/issues) や [フォーム](/form) でお知らせください。
